
# Type 1 Diabetes Daily Monitoring Dashboard

This project is a healthcare analytics portfolio project built with **SQL, PostgreSQL, Python, Pandas, Jupyter Notebook, Excel, and Tableau Public**. It analyzes synthetic Type 1 diabetes patient, glucose, and insulin data to support patient-level monitoring and dashboard reporting.

## Features
- relational healthcare data model
- synthetic glucose and insulin event generation
- SQL reporting views for daily patient metrics
- Tableau Public dashboard for trend monitoring
- Excel-based QA and validation tracking

## Key Metrics
- Average Glucose
- Time in Range (TIR)
- Low/High Reading Counts
- Insulin Event Count
- Total Daily Dose Units
- Basal vs Bolus Insulin Totals

## Tech Stack
SQL, PostgreSQL, Python, Pandas, Jupyter Notebook, Excel, Tableau Public
